Topic: Logarithmic Function

Logarithmic Function Definition

In mathematics, the logarithmic function is an inverse function to exponentiation. It is defined as:

For x > 0, a > 0, and a ≠1, y = loga x if and only if x = a^y. The function is given by f(x) = loga x, where the base of the logarithm is 'a'.

The most common bases used in logarithmic functions are base 10 (common logarithms) and base e (natural logarithms).

Common Logarithmic Function

The logarithmic function with base 10 is called the common logarithmic function and is denoted by log10 or simply log.

For example, f(x) = log10 x.

Natural Logarithmic Function

The logarithmic function with base e is called the natural logarithmic function and is denoted by loge.

For example, f(x) = loge x.

Logarithmic Functions Properties

Logarithmic functions have several important properties that simplify calculations, especially when dealing with products, quotients, or powers. Here are the main properties:

  • Product Rule: logb(MN) = logb(M) + logb(N) – Multiply two numbers with the same base and add the exponents.
  • Quotient Rule: logb(M/N) = logb(M) – logb(N) – Divide two numbers with the same base and subtract the exponents.
  • Power Rule: logb(M^p) = p * logb(M) – Raise an exponential expression to a power and multiply the exponents.
  • Zero Exponent Rule: loga(1) = 0.
  • Change of Base Rule: logb(x) = ln(x) / ln(b) or logb(x) = log10(x) / log10(b).

Other Important Rules of Logarithmic Functions

Additional useful rules for logarithmic functions include:

  • logb(b) = 1 (e.g., log10(10) = 1)
  • logb(b^x) = x (e.g., log10(10^x) = x)
  • Exponentiation of base and logarithm: b^(logb(x)) = x.

Logarithmic Function Examples

Below are some examples that apply the logarithmic properties:

Example 1: Use the properties of logarithms to write the equation as a single logarithm:

5 log9(x) + 7 log9(y) - 3 log9(z) = log9(x^5) + log9(y^7) - log9(z^3) = log9((x^5 * y^7) / z^3)

Example 2: Use the properties of logarithms to write the equation as a single logarithm:

1/2 log2(x) - 8 log2(y) - 5 log2(z) = log2(x^1/2) - log2(y^8) - log2(z^5) = log2(x^1/2 / (y^8 * z^5))
